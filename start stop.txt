# Inventory Management System - Data & Input Requirements

## 1. PRODUCT DATA

### Required Fields:
- **Product Name**: Name of the product (e.g., "Laptop HP ProBook 450")
- **SKU**: Unique Stock Keeping Unit code (e.g., "LAP-HP-450-001")
- **Category**: Select from available categories
- **Quantity**: Current stock quantity (e.g., 50)
- **Price**: Selling price (e.g., 45000)
- **Cost**: Purchase/manufacturing cost (e.g., 35000)

### Optional Fields:
- **Description**: Detailed product description
- **Supplier**: Select from supplier list
- **Reorder Level**: Minimum stock before reorder alert (e.g., 10)
- **Status**: Active/Inactive/Discontinued

### Example Product Entry:
```json
{
  "name": "Dell Mouse Wireless",
  "sku": "ACC-DELL-MS-001",
  "description": "Wireless optical mouse with USB receiver",
  "category": "Electronics Accessories",
  "supplier": "Tech Distributors Ltd",
  "quantity": 150,
  "reorderLevel": 20,
  "price": 850,
  "cost": 600,
  "status": "active"
}
```

## 4. TRANSACTION DATA

### Required Fields:
- **Product**: Select product from list
- **Type**: Transaction type
  - `in` - Stock incoming/purchase
  - `out` - Stock outgoing/sale
  - `adjustment` - Stock correction
- **Quantity**: Number of units

### Optional Fields:
- **Reason**: Reason for transaction (e.g., "Customer order", "Damaged goods")
- **Reference**: Reference number (e.g., Invoice number, PO number)

### Example Transactions:
```json
[
  {
    "product": "Dell Mouse Wireless",
    "type": "in",
    "quantity": 100,
    "reason": "New stock purchase",
    "reference": "PO-2025-001"
  },
  {
    "product": "Dell Mouse Wireless",
    "type": "out",
    "quantity": 5,
    "reason": "Customer sale",
    "reference": "INV-2025-045"
  },
  {
    "product": "Dell Mouse Wireless",
    "type": "adjustment",
    "quantity": -2,
    "reason": "Damaged during storage",
    "reference": "ADJ-2025-003"
  }
]
```

## 7. DATA VALIDATION RULES

### Product:
- ✓ Name: Required, 3-100 characters
- ✓ SKU: Required, unique, alphanumeric with hyphens
- ✓ Quantity: Required, number ≥ 0
- ✓ Price: Required, number > 0
- ✓ Cost: Required, number > 0
- ✓ Reorder Level: Optional, number ≥ 0

### Supplier:
- ✓ Name: Required, 3-100 characters
- ✓ Email: Required, valid email format
- ✓ Phone: Required, 10-15 digits

### Transaction:
- ✓ Product: Required, must exist
- ✓ Type: Required, one of: in/out/adjustment
- ✓ Quantity: Required, number > 0
